# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

**Email**: security@uetuluk.com

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

### Response Timeline

- **Initial response**: Within 7 days
- **Status updates**: As investigation progresses
- **Resolution**: Depends on severity and complexity

## Scope

### In Scope

- The portfolio website at uetuluk.com
- AI prompt handling and input processing
- API endpoints and data handling

### Out of Scope

- Cloudflare infrastructure and platform services
- Third-party services (OpenRouter, etc.)
- Social engineering attacks
- Denial of service attacks

## Acknowledgments

I appreciate responsible disclosure. Contributors who report valid vulnerabilities will be acknowledged here (with permission).
